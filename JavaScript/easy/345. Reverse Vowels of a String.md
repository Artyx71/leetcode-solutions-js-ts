# 345. Reverse Vowels of a String

## Алгоритм (RU)

**Подход:** Сбор индексов гласных + обратная перестановка.

Находим все позиции гласных в строке, затем меняем символы на этих позициях в обратном порядке.

- **Сложность по времени:** O(n)
- **Сложность по памяти:** O(n)

**Шаги:**
1. `vowels` — Set из 10 гласных (строчные + заглавные)
2. `arr = s.split('')` — массив символов для мутации
3. `indices` — массив позиций, где стоят гласные
4. `reversed` — символы с этих позиций в обратном порядке
5. Расставляем `reversed[i]` обратно на позиции `indices[i]`
6. `arr.join('')` → результат

---

## Algorithm (EN)

**Approach:** Collect vowel indices + reverse swap.

Find all positions of vowels in the string, then place the characters at those positions in reverse order.

- **Time complexity:** O(n)
- **Space complexity:** O(n)

**Steps:**
1. `vowels` — Set of 10 vowels (lower + upper case)
2. `arr = s.split('')` — character array for mutation
3. `indices` — positions where vowels occur
4. `reversed` — characters at those positions in reverse order
5. Write `reversed[i]` back to `indices[i]`
6. `arr.join('')` → result
