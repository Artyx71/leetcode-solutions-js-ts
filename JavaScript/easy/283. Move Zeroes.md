# 283. Move Zeroes

## Алгоритм (RU)

**Подход:** Фильтрация с перезаписью массива на месте.

Отфильтровываем все ненулевые элементы, считаем количество нулей. Очищаем исходный массив, записываем обратно ненулевые элементы, затем добавляем нули в конец.

- **Сложность по времени:** O(n)
- **Сложность по памяти:** O(n) — временный массив `filtered`

**Шаги:**
1. `filtered = nums.filter(el => el !== 0)` — убираем нули
2. `zeroCount = nums.length - filtered.length` — считаем нули
3. `nums.length = 0` — очищаем исходный массив на месте
4. `nums.push(...filtered)` — возвращаем ненулевые элементы
5. Добавляем `zeroCount` нулей в конец

> `nums.length = 0` изменяет тот же массив (мутация), не создаёт новый — требование задачи.

---

## Algorithm (EN)

**Approach:** Filter and in-place overwrite.

Filter out all non-zero elements, count the zeros. Clear the original array, push non-zero elements back, then append zeros at the end.

- **Time complexity:** O(n)
- **Space complexity:** O(n) — temporary `filtered` array

**Steps:**
1. `filtered = nums.filter(el => el !== 0)` — remove zeros
2. `zeroCount = nums.length - filtered.length` — count zeros
3. `nums.length = 0` — clear the original array in-place
4. `nums.push(...filtered)` — restore non-zero elements
5. Append `zeroCount` zeros to the end

> `nums.length = 0` mutates the same array reference — required by the problem constraints.
