# 461. Hamming Distance

## Алгоритм (RU)

**Подход:** XOR + подсчёт единичных битов.

XOR двух чисел даёт 1 на тех позициях, где биты различаются. Переводим результат в двоичную строку и считаем количество `'1'`.

- **Сложность по времени:** O(log n) — длина двоичного представления
- **Сложность по памяти:** O(log n) — строка битов

**Шаги:**
1. `x ^ y` — XOR: бит = 1 там, где x и y различаются
2. `.toString(2)` — двоичная строка
3. `.split("").filter(bit => bit === "1").length` — считаем единицы

---

## Algorithm (EN)

**Approach:** XOR + count set bits.

XOR of two numbers produces 1 at every bit position where they differ. Convert result to binary string and count `'1'` characters.

- **Time complexity:** O(log n) — length of binary representation
- **Space complexity:** O(log n) — binary string

**Steps:**
1. `x ^ y` — XOR: bit = 1 where x and y differ
2. `.toString(2)` — binary string
3. `.split("").filter(bit => bit === "1").length` — count the ones
