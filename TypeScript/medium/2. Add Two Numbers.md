# 2. Add Two Numbers

## Алгоритм (RU)

**Подход:** Посимвольное сложение связных списков с переносом (carry).

Оба списка представляют числа в обратном порядке. Проходим по обоим спискам одновременно, суммируем цифры и перенос. Если сумма ≥ 10 — сохраняем остаток, перенос = 1.

- **Сложность по времени:** O(max(m, n)) — m и n длины списков
- **Сложность по памяти:** O(max(m, n)) — для результирующего списка

**Шаги:**
1. Создаём фиктивный головной узел
2. Идём по обоим спискам пока есть узлы или ненулевая сумма
3. Добавляем значения `l1.val` и `l2.val` к `sum`
4. Если `sum >= 10`: `carry = 1`, `sum -= 10`
5. Создаём новый узел с `sum`, переходим к следующим узлам
6. Следующая итерация начинается с `sum = carry`

---

## Algorithm (EN)

**Approach:** Digit-by-digit addition of linked lists with carry.

Both lists represent numbers in reverse order. Traverse both lists simultaneously, summing digits and carry. If sum ≥ 10 — keep the remainder, carry = 1.

- **Time complexity:** O(max(m, n)) — m and n are list lengths
- **Space complexity:** O(max(m, n)) — for the result list

**Steps:**
1. Create a dummy head node
2. Loop while either list has nodes or sum > 0
3. Add `l1.val` and `l2.val` to `sum`
4. If `sum >= 10`: `carry = 1`, `sum -= 10`
5. Create new node with `sum`, advance to next nodes
6. Next iteration starts with `sum = carry`
