# 1. Two Sum

## Алгоритм (RU)

**Подход:** Полный перебор с фильтрацией.

Для каждого элемента `nums[i]` ищем в массиве другой элемент, равный `target - nums[i]`. Используем `filter`, чтобы найти дополнение, исключая текущий индекс. Если находим — возвращаем оба индекса.

- **Сложность по времени:** O(n²) — вложенный перебор
- **Сложность по памяти:** O(1)

**Шаги:**
1. Перебираем каждый `nums[i]`
2. Фильтруем массив: ищем элемент равный `target - nums[i]`, не совпадающий по индексу
3. Если результат найден — берём индекс второго элемента через `indexOf`, начиная с `i+1`
4. Возвращаем `[i, secondIndex]`

---

## Algorithm (EN)

**Approach:** Brute force with filtering.

For each element `nums[i]`, search the array for another element equal to `target - nums[i]`. Use `filter` to find the complement, excluding the current index. If found — return both indices.

- **Time complexity:** O(n²) — nested iteration
- **Space complexity:** O(1)

**Steps:**
1. Iterate over each `nums[i]`
2. Filter the array for an element equal to `target - nums[i]` with a different index
3. If found — get the second element's index via `indexOf` starting from `i+1`
4. Return `[i, secondIndex]`
